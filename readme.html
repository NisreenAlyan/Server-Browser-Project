About
Simple Browser is a free web browser that is east to use. It takes away the excess features of a web browser and shows a stripped down version! It was originally intended for developers who needed users to find a page on the internet but is now a web browser for everyday use as well. Simple Browser is desperately trying to remain free and high quality. However this is hard without the help of donations.

System Requirements
Windows
  • Windows 2000/ME/XP/VISTA
  • 1 MB of free space (or more)
  • 256 MB of RAM (or more)
  • Internet Explorer 6 (or above)
Linux/Mac
  • Linux OS/Mac: 10.2 (or above)
  • A 5th Generation (or above)
  • 1 MB of free space (or more)
  • 256 MB of RAM (or more)

Installation
Windows
  1. Download Simple Browser here.
  2. Extract the files to your desktop.
  3. Run Simple Browser.hta by double-clicking on it.
  4. Surf The Web With Ease!
Linux/Mac
  1. Download Simple Browser Source here.
  2. Extract the files to your desktop.
  3. Run simplebrowser.html by double-clicking on it.
  4. Hide the toolbars in your normal web browser.
  5. Surf The Web With Ease!

The Interface
The interface in Simple Browser is plain, basic and text-based.

Print


Print

This opens the Print Dialog which allows you to print the page.

Back


Back

This goes back one (1) page from where you were.

Forward


Forward

This would go forward one (1) page if you had gone back.

Home


Home

This leads to the homepage. The homepage describes Simple Browser.

Refresh


Refresh

This refreshes the current page.

View Source


View Source

This shows the page source (code to display webpage).

About


About

This shows the license (copyright) and version information.

URL Box




This allows you to enter a URL (web address) and then be directed to it in Simple Browser.

Errors
When surfing the web, you will encounter errors.
If Simple Browser does not change pages, you have entered the URL incorrectly.
For a list of errors you may stumble upon when browsing can be found here.
